# HKU-FOSS-Thesis-Template
Latex Template for RPG thesis in Faculty of Social Science (Unofficial)

This project is based on [HKU-Thesis-Template](https://github.com/monaen/HKU-Thesis-Template/tree/master) and has been modified to align with the style of the Faculty of Social Sciences (FOSS).

## Modifications

The main modifications made in this project include:

- **Cover Page Design**: The layout and style of the cover page have been adjusted to meet the requirements of the FOSS.
- **Content Arrangement**: The order of elements from the cover page to the publication list has been reorganized.
- **Font Usage**: Times New Roman font has been adopted to comply with the required font specifications.
- **Citation Style**: The in-text citation style follows APA 7th edition guidelines to ensure academic compliance.
- **Header Settings**: The headers have been set to display the chapter titles for easier navigation.
- **Page Number Position**: Page numbers are placed at the bottom of the page to align with formatting standards.
